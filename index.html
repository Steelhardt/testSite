What Did I learn?

I learned that creating objects in a scene with 3js is easy to do. Although creating animations like the scrolling effect you see now was challenging to understand.

So how do you make the scrolling animation?

In order to create the scroll animation you need to go into styles.css and make

  • main{position:absolute //to make it not move//
  • display: grid; //this is to create sections with scrolling
  • grid-template-columns: repeat(12, 1fr); /*to make 12 even columns that we can access*/
  • For example in styles.css header { background: var(--dark-bg); grid-column: 2 / span 5; /*START on col 2 span 5 columns*/}y
